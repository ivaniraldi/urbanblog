# 🏙️ UrbanBlog

**UrbanBlog** es un blog desarrollado como prueba para [WebRush Brasil](https://webrushbrasil.com.br), una empresa de marketing y desarrollo web. Construido con **Next.js, TailwindCSS, React y Framer Motion**, este blog permite a los usuarios:

✅ **Comentar posts** 📩
✅ **Visualizar publicaciones** 📰
✅ **Contactar con WebRush Brasil** 💼
✅ **Explorar la sección de blogs destacados** ⭐

## 🚀 Tecnologías utilizadas
- ⚡ **Next.js** - Para una experiencia de navegación rápida y optimizada.
- 🎨 **TailwindCSS** - Diseño moderno y responsivo.
- 💫 **Framer Motion** - Animaciones fluidas y atractivas.
- ⚛️ **React** - Para una experiencia de usuario dinámica.

## 🛠 Instalación

Para ejecutar el proyecto en tu entorno local, sigue estos pasos:

### 1️⃣ Clonar el repositorio
```bash
git clone https://github.com/ivaniraldi/urbanblog.git
cd urbanblog
```

### 2️⃣ Instalar dependencias
```bash
npm install
```

### 3️⃣ Ejecutar la aplicación
```bash
npm run dev
```
